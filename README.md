# CSM - A simple C++ Code Snippet Manager

CSM gets command line arguments and adds them to files (currently .txt)

## Features:
 - Snippet adding - append snippet at the end of file
 - Language adding - allows the user to write what language the snippet is in
 - Tag adding - users can add as many tags as they like to the snippet
